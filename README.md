# 🚀 100 Days of DSA Challenge

Welcome to my **100 Days of Data Structures and Algorithms (DSA) Challenge**!

I started this challenge to improve my problem-solving skills, strengthen my understanding of DSA concepts, and prepare for coding interviews and placements.

## 📌 About

In this repository, I will be solving DSA problems daily using **Java**. The problems are taken from platforms like:
- LeetCode
- GeeksforGeeks
- College DSA Assignments

Each day's solutions are organized separately to keep my progress structured and easy to follow.
